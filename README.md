# fabula

**Phrase-based authentication gate.**

A person enters not with a password, but with a phrase — a sentence, a reference, a line from a story.
The gate validates the phrase against a known pattern and grants or denies access.

## Concept

Traditional auth: you know a secret → enter.
Fabula: you know a *context* → the phrase proves you belong.

The phrase is human-readable. It can be spoken, written, remembered, or shared as a story.
The gate is deterministic: the same phrase always maps to the same identity.
